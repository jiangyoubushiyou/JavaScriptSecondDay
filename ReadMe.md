# 常见的HTML事件
`onchange`-HTML元素改变  
`onclick`-用户点击HTML元素  
`onmouseover`-用户在一个HTML元素上移动鼠标
`onmouseout`-用户在一个HTML元素上移开鼠标   
`onkeydown`-用户按下键盘按键   
`onload`-浏览器已完成页面的加载
